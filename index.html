1. Inicio e configuracao

git init Cria um repositorio Git local.

git clone URL Baixa um repositorio remoto.

git config --global user.name "Seu Nome" Define nome global.

git config --global user.email "email@dominio.com" Define email global.

git config --list Lista configuracoes ativas.

git config --global init.defaultBranch main Define branch padrao.

2. Fluxo diario

git status Mostra estado dos arquivos.

git add arquivo Adiciona arquivo ao staging.

git add . Adiciona alteracoes do diretorio atual.

git commit -m "mensagem" Cria commit com mensagem.

git commit --amend Ajusta ultimo commit.

git mv origem destino Move/renomeia arquivo rastreado.

git rm arquivo Remove arquivo e registra remocao.

git restore arquivo Restaura arquivo do indice/HEAD.

git restore --staged arquivo Retira do staging.

Voltar ao topo

3. Branches e integracao

git branch Lista branches locais.

git branch nome Cria nova branch.

git switch nome Troca para branch existente.

git switch -c nome Cria e troca para nova branch.

git checkout nome Forma antiga para trocar branch/arquivo.

git merge branch Junta branch na atual.

git rebase branch Reaplica commits sobre outra base.

git cherry-pick hash Aplica commit especifico.

git branch -d nome Remove branch mesclada.

git branch -D nome Forca remocao de branch.

Voltar ao topo

4. Comandos de merge

git merge nome-da-branch Mescla a branch informada na branch atual.

git merge --no-ff nome-da-branch Forca commit de merge, mantendo historico explicito.

git merge --ff-only nome-da-branch So permite fast-forward, sem commit de merge.

git merge --squash nome-da-branch Junta alteracoes em um unico commit manual.

git merge --no-commit nome-da-branch Prepara merge sem commitar automaticamente.

git merge -m "mensagem" nome-da-branch Define mensagem do commit de merge.

git merge --abort Cancela merge em conflito e volta ao estado anterior.

git merge --continue Continua merge apos resolver conflitos.

git mergetool Abre ferramenta para resolver conflitos visualmente.

git log --merges Lista somente commits de merge.

Voltar ao topo

5. Repositorios remotos

git remote -v Lista remotos configurados.

git remote add origin URL Adiciona remoto.

git fetch Busca novidades sem mesclar.

git pull Busca e integra (fetch + merge/rebase).

git pull --rebase Atualiza com rebase.

git push Envia commits para remoto.

git push -u origin branch Define upstream.

git push --force-with-lease Force push seguro.

git remote rename antigo novo Renomeia remoto.

git remote remove nome Remove remoto.

Voltar ao topo

6. Historico e comparacao

git log Mostra historico de commits.

git log --oneline --graph --decorate --all Historico resumido em grafo.

git show hash Exibe detalhes de commit/tag.

git diff Diferencas nao staged.

git diff --staged Diferencas staged.

git diff A..B Compara dois pontos de historico.

git blame arquivo Mostra autoria por linha.

git shortlog -sn Resumo de contribuicoes.

git reflog Historico de movimentos do HEAD.

Voltar ao topo

7. Desfazer e recuperar

git reset --soft HEAD~1 Desfaz commit mantendo staging.

git reset --mixed HEAD~1 Desfaz commit e limpa staging.

git reset --hard HEAD~1 Desfaz commit e descarta alteracoes locais.

git revert hash Cria commit de reversao sem reescrever historico.

git checkout -- arquivo Forma antiga de restaurar arquivo.

git restore --source=HEAD~1 arquivo Recupera arquivo de revisao anterior.

Voltar ao topo

8. Stash e limpeza

git stash Guarda alteracoes locais temporariamente.

git stash push -m "nota" Cria stash com descricao.

git stash list Lista stashes.

git stash pop Aplica e remove stash.

git stash apply Aplica sem remover stash.

git clean -n Simula limpeza de arquivos nao rastreados.

git clean -fd Remove arquivos/pastas nao rastreados.

Voltar ao topo

9. Tags e releases

git tag Lista tags.

git tag v1.0.0 Cria tag leve.

git tag -a v1.0.0 -m "release" Cria tag anotada.

git show v1.0.0 Exibe dados da tag.

git push origin v1.0.0 Publica tag especifica.

git push origin --tags Publica todas as tags.

git tag -d v1.0.0 Remove tag local.

Voltar ao topo

10. Submodulos e subarvores

git submodule add URL caminho Adiciona submodulo.

git submodule update --init --recursive Inicializa/atualiza submodulos.

git submodule foreach 'git pull' Atualiza cada submodulo.

git subtree add --prefix=pasta URL main --squash Importa projeto como subtree.

git subtree pull --prefix=pasta URL main --squash Atualiza subtree.

11. Manutencao e desempenho

git gc Coleta lixo e otimiza objetos.

git fsck Verifica integridade do repositorio.

git prune Remove objetos nao acessiveis (uso com cuidado).

git repack -a -d Reempacota objetos para eficiencia.

git count-objects -vH Mostra estatisticas de objetos.

git maintenance run Executa manutencao automatizada.

12. Comandos internos (plumbing)

Comandos mais tecnicos, usados em scripts, ferramentas e depuracao.

git hash-object arquivo Gera hash do objeto.

git cat-file -p hash Le objeto Git pelo hash.

git ls-files Lista arquivos rastreados no indice.

git ls-tree -r HEAD Lista arvore de arquivos de um commit.

git rev-parse HEAD Resolve referencias para hashes.

git update-index Manipula indice diretamente.

git write-tree Escreve arvore do indice.

git commit-tree Cria commit manual a partir de tree.

git mktree Cria objeto tree manualmente.

git unpack-objects Expande objetos de packfiles.

git pack-objects Cria packfiles.

git verify-pack Verifica conteudo de pack.
